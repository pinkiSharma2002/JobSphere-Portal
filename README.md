# JobSphere Portal

## Overview
JobSphere Portal is a web-based recruitment management system that helps connect job seekers and recruiters. The platform allows recruiters to post job openings, manage applications, and find suitable candidates, while job seekers can search for jobs and apply online.

## Features
- User Registration & Login
- Recruiter and Job Seeker Modules
- Job Posting & Management
- Job Search & Filtering
- Profile Management
- Secure MySQL Database

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Software Requirements
- XAMPP/WAMP
- PHP 8+
- MySQL
- Web Browser
